    How to use the extension?

    Shortcut to a page:

    1. Launch the extension by typing qq in the address bar and pressing space.

    2. Type the shortcut name and press Enter.

    Search using a shortcut: (only pre-built shortcut support)

    1. Launch the extension by typing qq in the address bar and pressing space.

    2. Type !shortcut_name.

    3. Type your search query and press Enter.

    Enjoy using the extension!
